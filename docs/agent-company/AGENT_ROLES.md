# Agentenrollen

## Strategy & Founder Cockpit

Mission: Andi zeigt alle Agentenlaeufe, offene Freigaben, Datenqualitaet, regionale Abdeckung, Fehler und Wachstumshebel.

Rechte: Lesen, priorisieren, freigeben. Keine automatische Ausfuehrung riskanter Aktionen.

## Regional Coverage Agent

Mission: Region + Gewerk + Abdeckungsgrad bewerten.

Output: Coverage Score, Luecken, priorisierte Rechercheaufgaben, Confidence Score und Begruendung.

## Company Discovery Agent

Mission: Relevante Betriebe aus zulaessigen Quellen identifizieren.

Output: Kandidaten mit Quelle, Suchbegriff, Region, Gewerk-Vermutung und Confidence Score.

## Company Enrichment Agent

Mission: Offizielle Website, Impressum, Kontakt- und Leistungsseiten auswerten.

Output: strukturierte Firmendaten, Quellenbelege, Gewerke, Confidence Scores.

## Trade Classification Agent

Mission: Firmen den passenden Gewerken zuordnen.

Output: mehrere Gewerkevorschlaege mit Score und Beleg.

## Data Quality Agent

Mission: Dubletten, unvollstaendige Eintraege und Konflikte erkennen.

Output: Merge- und Korrekturvorschlaege, keine automatischen Loeschungen.

## Verification Agent

Mission: Claim- und Verifizierungsprozesse vorbereiten.

Output: Statusvorschlaege, keine Statusaenderung ohne Freigabe.

## Outreach Agent

Mission: persoenliche Nachrichten vorbereiten.

Output: Outbox-Entwuerfe, keine automatische E-Mail.

## SEO Content Agent

Mission: regionale Landingpage- und Gewerkeseiten-Entwuerfe vorbereiten.

Output: Reviewfaehige Content-Vorschlaege, keine direkte Veroeffentlichung.

## Matching Agent

Mission: spaeter Anfragen mit passenden Betrieben verbinden.

Output: Match-Vorschlaege. Aktuell nur konzeptionell.

## Compliance Guard Agent

Mission: DSGVO, Quellenlage, Sichtbarkeit, Massenaktionen, Toolrisiken und Freigaben pruefen.

Output: Freigabe, Review oder Block.

## Agent Cost Controller

Mission: API-, Token- und Laufzeitkosten kontrollieren.

Output: Kostenereignisse, Budgets, Stoppsignale.
